# Claude Instructions for Project

## Project Overview
สร้าง web app สำหรับจัดการธนาคารขยะภายในชุมชนขนาดเล็ก
ระบบจะบันทึกข้อมูลสมาชิก ยอดขยะสะสม และคำนวณเงินให้โดยอัตโนมัติ
สามารถส่ง SMS หรือแจ้งเตือนยอดเงินให้สมาชิกทราบได้
web app มีลงชื่อเข้าใช้มี 2 ระดับ ผู้ใช้ และ แอดมิน

## System Goals
- จัดการข้อมูลสมาชิกของธนาคารขยะ
- บันทึกและติดตามยอดขยะสะสมของสมาชิก
- คำนวณเงินจากยอดขยะโดยอัตโนมัติ
- แจ้งเตือนยอดเงินให้สมาชิกทราบผ่าน SMS หรือระบบแจ้งเตือน
- รองรับการใช้งานทั้งบนมือถือและคอมพิวเตอร์
- ใช้ฐานข้อมูล SQLite
- มี Frontend และ Backend ที่เชื่อมต่อกัน

## Functional Requirements
- ระบบต้องมีระบบลงชื่อเข้าใช้
- ระบบต้องมีสิทธิ์ผู้ใช้ 2 ระดับ: ผู้ใช้และแอดมิน
- ระบบต้องบันทึกข้อมูลสมาชิก
- ระบบต้องบันทึกยอดขยะสะสมของสมาชิก
- ระบบต้องคำนวณเงินจากยอดขยะอัตโนมัติ
- ระบบต้องแสดงยอดขยะและยอดเงินให้ผู้ใช้เห็นได้
- ระบบต้องสามารถส่ง SMS หรือแจ้งเตือนยอดเงินให้สมาชิกทราบ
- ระบบต้องเก็บข้อมูลในฐานข้อมูล SQLite

## Non-Functional Requirements
- ระบบต้องทำงานได้เสถียรและรวดเร็ว
- ระบบต้องมีความปลอดภัยในการจัดการข้อมูลผู้ใช้
- ระบบต้องใช้งานง่ายและเข้าใจง่าย
- ระบบต้องรองรับอุปกรณ์หลายขนาดหน้าจอ
- ระบบต้องมีความน่าเชื่อถือในการจัดเก็บข้อมูล

## Constraints
- ใช้ฐานข้อมูล SQLite
- เหมาะสำหรับระบบขนาดเล็กถึงกลาง
- การแจ้งเตือน SMS อาจขึ้นอยู่กับบริการภายนอก

## Development Guidance
- ให้พัฒนาเป็น web application ที่มีทั้ง frontend และ backend
- ควรออกแบบระบบให้แยกแยะระหว่างบทบาทผู้ใช้และแอดมิน
- ให้ให้ความสำคัญกับความง่ายในการใช้งานและการจัดการข้อมูลสมาชิก
- ควรพิจารณาความปลอดภัยของข้อมูลการเข้าสู่ระบบและข้อมูลสมาชิก
