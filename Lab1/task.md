# Task List

## โครงการ: ระบบแจ้งเหตุเพลิงไหม้

### วัตถุประสงค์
สร้างเว็บแอปพลิเคชันต้นแบบเพื่อแสดงสถานะการตรวจจับควันและเปลวไฟ พร้อมระบบแจ้งเตือนและประวัติเหตุการณ์ที่อ่านง่าย

## Tasks

### 1. Setup โครงสร้างโปรเจกต์
- [ ] สร้างโปรเจกต์ Next.js พร้อมโครงสร้างพื้นฐาน
- [ ] เตรียมโฟลเดอร์สำหรับหน้า UI, API, components, services, types

### 2. พัฒนา UI สำหรับสถานะระบบ
- [ ] สร้างหน้าแรกแสดงสถานะการตรวจจับควันและเปลวไฟ
- [ ] แสดงสถานะปัจจุบันแบบชัดเจน (ปกติ / อันตราย / ไม่ทราบสถานะ)
- [ ] ออกแบบหน้าให้รองรับมือถือและเดสก์ท็อป

### 3. Implement ระบบแจ้งเตือน
- [ ] สร้างกลไกแสดงข้อความเตือนเมื่อเกิดเหตุการณ์ความเสี่ยง
- [ ] แสดงข้อความเตือนแบบเข้าใจง่ายและตรงกับสถานะเหตุการณ์
- [ ] จัดเตรียมโครงสร้างสำหรับการแจ้งเตือนในอนาคต

### 4. พัฒนาประวัติการแจ้งเตือน
- [ ] สร้างหน้า/ส่วนแสดงประวัติการแจ้งเตือนย้อนหลัง
- [ ] แสดงวันที่และเวลาที่เกิดเหตุการณ์
- [ ] จัดรูปแบบข้อมูลให้อ่านง่าย

### 5. API / Service Layer
- [ ] สร้าง API endpoint สำหรับสถานะระบบ
- [ ] สร้าง API endpoint สำหรับประวัติการแจ้งเตือน
- [ ] แยก business logic ออกจาก UI component

### 6. Quality Assurance
- [ ] ตรวจสอบว่า UI ทำงานได้ถูกต้องบนอุปกรณ์ต่าง ๆ
- [ ] รัน build และ lint เพื่อยืนยันความเสถียร
- [ ] ทบทวนความครบถ้วนตาม SRS

## Deliverables
- หน้าแสดงสถานะระบบ
- ระบบแจ้งเตือนพื้นฐาน
- หน้าประวัติการแจ้งเตือน
- เอกสาร/โค้ดที่สามารถต่อยอดได้ในอนาคต

## Priority
1. High: UI สถานะระบบ
2. High: ระบบแจ้งเตือน
3. Medium: ประวัติการแจ้งเตือน
4. Medium: การปรับปรุง UX และความเสถียร
